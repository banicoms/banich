# Banich (Version 3.0) 🚀

**Banich** یک افزونه مرورگر متن‌باز است که برای بهبود خوانایی وب‌سایت‌ها طراحی شده است. این افزونه با اعمال فونت محبوب و استاندارد **وزیر (Vazirmatn)** و اجبار چیدمان **راست‌به‌چپ (RTL)**، گشت‌وگذار در اینترنت را برای فارسی‌زبانان لذت‌بخش‌تر می‌کند.

---

## ویژگی‌های کلیدی | Features
* **تغییر خودکار فونت:** جایگزینی فونت‌های پیش‌فرض وب‌سایت‌ها با فونت خوانای "وزیر".
* **پشتیبانی از RTL:** اصلاح چیدمان متن در سایت‌هایی که از راست‌به‌چپ پشتیبانی نمی‌کنند.
* **بهینه‌سازی نسخه ۳:** هماهنگی کامل با Manifest V3 گوگل کروم برای امنیت و سرعت بیشتر.
* **سادگی در استفاده:** بدون نیاز به تنظیمات پیچیده، بلافاصله پس از نصب فعال می‌شود.
